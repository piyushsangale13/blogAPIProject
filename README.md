## About

The "Blog Website" project is a web application that allows users to create, read, update, and delete blogs. It provides a platform for users to share their thoughts and experiences through blog posts.

## Features

- **Create:** Users can create new blog posts by providing a title, content, and other relevant details.
- **Read:** Visitors can read existing blog posts to explore a variety of topics.
- **Update:** Authors can edit and update their published blog posts.
- **Delete:** Users have the option to delete their blog posts if needed.



## Installation

1. Clone or download the project from the GitHub repository:

```bash
git clone https://github.com/piyushsangale13/blogAPIProject.git
```

2. Navigate to the project directory:

```bash
cd blogAPIProject
```

3. Install dependencies:

```bash
npm install
```

## Usage


1. **Run the Server:**
   Start the server by running the following command in the terminal.

   ```bash
   node ./index.js
   ```

2. **Start the Application:**
   Open a new terminal and run the following command to start the application.

   ```bash
   node ./server.js
   ```
3. **Access the Application:**
   Open your web browser and go to [http://localhost:3000](http://localhost:3000) to access the Blog Website.
